# schmordle-the-rhyming-wordle-app
“Schmordle” is a twist on the classic NYT Wordle game, where players must guess any five-letter word from a special set, except in our version, all the correct answers rhyme with each other.
## Modes
- Solitaire Mode
- Real-Time Multiplayer
- Daily Challenge

## Tech Stack
- React (frontend)
- Node.js / Express (backend)
- Firebase Auth + Firestore
- WebSockets (real-time multiplayer)

## Setup

### Client
cd client  
npm install  
npm run dev  

### Server  
cd server  
npm install  
npm start
